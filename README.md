# 🍜 大学城美食地图 Campus Food Map

专为高校学生打造的本地化美食发现与社交平台，以「吃什么」为核心，融合场景推荐、榜单排行、饭搭子社交与成就体系，让每一餐都不再纠结。

## ✨ 核心功能

- **场景化推荐**：按用餐人数、餐品类型、预算、饮食计划多维筛选，一键随机推荐
- **美食榜单**：综合榜、好评榜、性价比榜 + 美食家达人榜，支持按菜系细分筛选
- **饭搭子社交**：基于口味偏好智能匹配饭搭子，支持打招呼、私聊、饭友圈 UGC 分享
- **打卡 & 成就体系**：到店打卡记录用餐轨迹，30+ 趣味成就（含隐藏成就）等你解锁
- **个性化建档**：首次使用采集饮食偏好，持续学习用户行为，越用越懂你
- **学生福利**：专属优惠券、满减活动，学生认证享更多权益
- **隐私优先**：社交功能需主动开启，数据可见性精细控制

## 🛠 技术栈

- **框架**：React 18 + Vite
- **样式**：Tailwind CSS
- **路由**：React Router v6
- **状态管理**：React Context + LocalStorage 持久化
